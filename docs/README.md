# Documentacao do Projeto

Esta pasta reune os documentos usados durante o desenvolvimento e na entrega academica do projeto.

## Essenciais para entrega

- `relatorio-tecnico.md`: relatorio final consolidado.
- `video-demonstracao.md`: local para registrar o link publico do video.

## Materiais de apoio

- `arquitetura-diagrama.md`: resumo auxiliar da arquitetura.
- `checklist-requisitos.md`: checklist resumido dos requisitos da disciplina.
- `Checklist de Entrega.md`: checklist interno de acompanhamento da entrega.
- `Desenvolvimento de Software em Nuvem.md`: copia do enunciado/orientacoes da atividade.

## Observacao

Para submissao, o documento principal desta pasta e `relatorio-tecnico.md`. Os demais arquivos servem como apoio, validacao e organizacao interna da equipe.
