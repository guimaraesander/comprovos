# Checklist de Requisitos — ComprovOS

## Funcionais (Obrigatórios)
- [ ] Autenticação e autorização
- [ ] API RESTful documentada (Swagger/OpenAPI)
- [ ] CRUD completo
- [ ] Validação de dados no back-end
- [ ] Logs de acesso e erro

## Arquitetura Técnica (Obrigatórios)
- [ ] Front-end com framework moderno (React)
- [ ] Deploy do front-end em nuvem
- [ ] Back-end REST containerizado com Docker
- [ ] Deploy do back-end em nuvem
- [ ] Banco gerenciado em nuvem (persistência fora do container)

## DevOps (Obrigatórios)
- [ ] Docker no back-end
- [ ] CI/CD com build
- [ ] CI/CD com testes automatizados
- [ ] CI/CD com deploy automático (ou workflow preparado)

## Segurança e Boas Práticas
- [ ] Variáveis de ambiente
- [ ] Proteção de rotas
- [ ] Tratamento de erros
- [ ] Separação dev/prod (quando possível)

## Testes / Qualidade
- [ ] Testes de API no back-end
- [ ] Pelo menos 1 teste no front-end
- [ ] Evidência de execução / cobertura

## Colaboração
- [ ] Repositório público no GitHub
- [ ] Branches por funcionalidade
- [ ] Commits semânticos
- [ ] Issues/Kanban (GitHub Projects)

## Entregáveis
- [ ] README detalhado
- [ ] Relatório técnico (máx. 6 páginas)
- [ ] Vídeo até 7 minutos
